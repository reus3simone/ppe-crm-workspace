PPE客户开发工作区 — 文件功能索引
====================================

┌─ app.py（80行）
   入口：页面路由 / session 初始化 / 侧边栏导航

├─ pages/  ——— 页面层（Streamlit UI）
│
│  ├─ home.py（248行）
│  │     首页仪表盘
│  │     - 概览统计（总客户/A级/推进中/逾期）
│  │     - 开发阶段分布卡片（可点击跳转）
│  │     - 今天必须处理（逾期/到期客户）
│  │     - 次优先列表（3天内到期/变凉中）
│  │
│  ├─ workspace.py（563行）
│  │     客户工作台（核心页面）
│  │     - 客户列表（搜索/筛选/分页）
│  │     - 客户详情（信息/邮箱/时间线）
│  │     - 新建 / 编辑客户表单
│  │     - AI开发信生成
│  │     - 跟进记录 / 邮件历史
│  │     - 时区建议 & 文化指南
│  │     - 产品匹配推荐
│  │
│  ├─ research.py（109行）
│  │     背调工具
│  │     - 客户网站/产品调研记录
│  │
│  └─ settings.py（175行）
│         设置页
│         - 团队成员管理
│         - 撞客户保护名单
│         - 系统设置（公司介绍/AI提示词/SOP规则）
│         - 数据备份 & 恢复
│
└─ database/  ——— 数据层
   │
   ├─ db.py（1260行）
   │     Database 类（对外接口）
   │     - 建表 & schema 迁移
   │     - 客户 CRUD（增删改查）
   │     - 冲突检测（重复/疑似/联系人撞车）
   │     - 批量导入 Excel
   │     - 跟进时间线 & 邮件历史
   │     - 邮件模板管理
   │     - 系统设置读写
   │     - 团队成员管理
   │     - 撞客户保护名单
   │     - 全局活动流
   │     - SOP下次跟进计算（根据邮件轮次）
   │     - 时区建议 & 文化指南
   │     - 产品匹配推荐
   │     - 备份 & 恢复
   │
   ├─ data.py（336行）
   │     静态数据（纯字典/列表，无逻辑）
   │     - PRODUCT_RULES：产品匹配关键词表
   │     - TIMEZONE_MAP：各国时区 & 夏令时
   │     - CULTURE_GUIDE：14国商务文化指南
   │     - CN_COUNTRY_MAP：中文→英文国家名映射
   │
   └─ utils.py（139行）
        无状态工具函数（不依赖 Database 类）
        - extract_domain()：从邮箱/网址提取域名
        - calculate_auto_score()：客户自动评分
        - get_customer_grade_by_score()：分数→等级
        - get_customer_health()：跟进健康度判断
        - is_dst_season()：是否夏令时季节
        - add_working_days()：增加工作日
        - parse_contacts()：从文本解析联系方式
